Antes De Apuntarte

Deja todo preparado para cuando se abra la siguiente ronda.

Si el torneo vuelve a abrir plazas, te ayudara tener lista tu informacion basica y saber como funciona el formato.

  1. Lee las reglas para entender el sorteo de equipos y las composiciones aleatorias.
  2. Ten a mano tu ID del juego, tu usuario de Discord y tus roles principal y secundario.
  3. Usa la lista de espera para dejar constancia de interes y estar preparado para la proxima edicion.
